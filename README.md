# AVN Legal Portfolio

Site portfolio premium para um escritorio de advocacia empresarial ficticio.

## Estrutura

- `index.html` e paginas institucionais
- `assets/` com CSS, JS, imagens e video
- `scripts/generate-media.mjs` para regenerar midia IA

## Uso local

Abra a pasta no WAMP ou sirva os arquivos estaticos.

## Observacoes

- Conteudo, equipe, casos e contatos sao ficticios.
- O envio do formulario ainda e demonstrativo.
